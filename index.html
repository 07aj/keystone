Welcome to Keystone

Node.js content management system and application platform built on express and mongoose.

Get keystone from npm to use it in your projects:

npm install keystone

or view the project on GitHub.

For demos and examples, visit demo.keystonejs.com.

What does it do?

Keystone provides:

  • A simple way to create an Express web app with custom routes, templates and models
  • Out of the box session management and authentication
  • Enhanced models with additional field types and functionality, building on those natively supported by Mongoose
  • Tools to simplify creating and managing routes, views and actions in your application
  • An updates framework for managing data updates or initialisation
  • An auto-generated Admin UI based on the defined models
  • Useful utils for simplifying app development
  • Integration with Coudinary for image uploading, storage and resizing
  • Integration with Mandrill for sending emails easily
